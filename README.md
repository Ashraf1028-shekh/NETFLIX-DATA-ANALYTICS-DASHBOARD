🎬 Netflix Data Analytics Dashboard
📌 Project Overview

This project is a Netflix Data Analytics Dashboard built to analyze and visualize Netflix content data in an interactive and user-friendly way. It helps users explore trends, patterns, and insights from Netflix movies and TV shows.

The dashboard allows filtering, comparison, and exploration of content based on various attributes like genre, release year, ratings, and more.

🚀 Features
📊 Interactive dashboard with modern UI
🔍 Filter content by:
Genre
Release Year
Type (Movie / TV Show)
📈 Visual insights:
Content distribution
Top genres
Year-wise growth
💡 User-friendly design for quick understanding
⚡ Fast and lightweight (built using Python + Streamlit)
🛠️ Tech Stack
Python
Streamlit
Pandas
Matplotlib / Plotly

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/netflix-dashboard.git
cd netflix-dashboard
2. Install dependencies
pip install -r requirements.txt
3. Run the application
streamlit run app.py
📊 Dataset
The dataset used is Netflix Titles Dataset (CSV)
Make sure the dataset file is present in the project folder
If not included, upload your CSV file before running
